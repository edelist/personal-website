Multimodal Digital Clock

The multimodal digital clock has four modes: 24-hour clock, 12-hour clock, stopwatch and timer. It is designed for the Nexys A7 Artix-7 FPGA. Modes are toggled between using two switches acting as a 2-bit select line for a mode-select multiplexer. Please reference the constraints file for the programming of the buttons and switches. When FPGA is connect to a monitor via VPA port, the display reads in the format: hh:mm:ss

Arithmetic Login Unit

The ALU is described using behavioural Verilog and has four modes: adder, subtractor, shifter and concatenator - all are toggled between using a multiplexer. The four modes are toggled between with a 4-to-1 multiplexer. Each of the modes can receive a 4-bit input, whereas the concatenator receivces an 8-bit input.

Counter, Debouncer, 7-Segment Display Driver

There are two modes for counting: manual counting using button presses, and counting every second. These modes are toggled using a switch on the FPGA; please reference the contraints file for the programming of all buttons and switches. This hardware is described for the Nexys A7 Artix-7 FPGA. Manual Counting Mode The manual debouncer and counter work to increment the count every time the button is pressed, and will only increment once if the button is held down. Automatic Counting (1 second) The count will increment every second.

7-Segment Display The display on the FPGA shows the count in hexadecimal. The count syncs across both modes and will only reset upon presseing the global reset button on the FPGA.

Smart Bike Light

The bike light two 3V LEDs for a front light and one 3V LED for the brake light. The front lights adjust the brightness as sensed by the BH1750 light sensor. The brake light is automatically a solid red and blinks when the Nano BLE 33 senses movement. The light is powered through 4 recharable 18650 3.7V lithium-ion batteries. Its housing is constructed from laser cutting clear acrylic.

Physics Research Paper

We investigated how and why the physical properties of underwound and overwound guitar pickups (change in windings of the electromagnet) impacted the tonal quality. We originally hypothesized the correlation was due to changes in electromotive force, but found through analysis that inductance was altering the resonant frequency of the LC-circuit - responsible for the auditory discrepancies. We derived a theoretical relationship between the fast-Fourier transform (FFT) analysis and properties of electromagnetic induction with resonant frequencies of LC-circuits, using electrical impedance, inductive reactance, and capacitive reactance.

The Effect of Changing a Guitar's Pickup Configuration on Tonal Quality
